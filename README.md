# Numivas Flutter App

This is a Flutter port of the Numivas luxury coin collection management application.

## Getting Started

Specially designed for comprehensive investment-grade numismatic portfolios.

### Prerequisites

- Flutter SDK (Latest Stable)
- Dart SDK

### Running the App

1.  **Get Dependencies**
    ```bash
    flutter pub get
    ```

2.  **Run**
    ```bash
    flutter run
    ```

### Features

-   **Vault Dashboard**: View your collection value and active displays.
-   **Gallery**: Browse available displays, art backings, and accessories.
-   **Concierge**: Connect with numismatic experts.
-   **Support**: Installation guides and AI technical desk.
-   **Product Detail**: Deep dive into specific units with Curator Insights.

### Architecture

-   **Views**: `lib/views/` contains all screen implementations.
-   **Models**: `lib/models/` contains data structures.
-   **Data**: `lib/data/` contains mock data constants.
-   **Theme**: Luxury dark/light theme adapted from the original design.
